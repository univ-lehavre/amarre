## Développement

### Prérequis

- Node.js + pnpm

### Configuration

Créer votre fichier d’environnement local à partir de l’exemple :

```bash
cp .env.example .env
```

Puis renseigner les variables dans `.env` (ne pas committer de secrets).

### Lancer en local

```bash
pnpm install
pnpm dev
```
